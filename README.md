# Dekatlas

Rocq を使用して巨大数論を機械化するプロジェクトです。

巨大数論を機械化するというのは、曖昧さがない形で記述して充分に信頼できるソフトウェアに検証させるということです。このことは数学において「形式化する」と呼ぶのが普通なのですが、日常の言葉としての「形式」は「体裁だけ取り繕って中身がない」という良くない意味も含んでいるので、一般の方々にも分かりやすいように本文書では「機械化する」と呼ぶことにします。

## Dekatlas という名前の由来

"Dekatlas" という名前は、巨大数論の地図帳としても、巨大数論を下から支えるプロジェクトとしても、発展してほしいと願い、名づけました。 "dekai" は「でかい」のローマ字綴りであり、巨大数論のことを意味します。 "atlas" は、「地図帳」を意味する英単語であり、ギリシャ神話に登場する天空を下から支えているアトラスという神の名前でもあります。 "dekai" と "atlas" を元にした混成語を作ることで、巨大数論の地図帳という意味と、巨大数論を下から支えるプロジェクトという意味を、二重に持たせています。

## 巨大数論とは

巨大数論とは、巨大数について研究する学問です。いつ頃から「巨大数論」という言葉があるのかは不明ですが、私の予想では 2 ちゃんねるにおいて巨大数についてのスレッドが立てられるようになった 2002 年頃から使われるようになったのではないかと思います。 P.N. AKIRA SANADA さんが作成した "A CHARMED QUARK" というサイトにおいて「巨大数論」が使われていたという記録が、 2003 年 12 月 26 日のアーカイブ（[リンク](https://web.archive.org/web/20031226011628/http://www.bekkoame.ne.jp/ha/hc17910/short8.htm)）に残っていますから、少なくとも 2003 年頃には使われていたようです。

英語において「巨大数論」と同じ意味の単語は "googology" です。 "googology" は "googol" と "-logy" を組み合わせた言葉です。 "[André Joyce](https://googology.fandom.com/ja/wiki/Andr%C3%A9_Joyce)" (巨大数研究 Wiki) と "[André Joyce](https://googology.fandom.com/wiki/Andr%C3%A9_Joyce)" (Googology Wiki) が説明しているところによると "googology" という言葉を考案したのは Michael Joseph Halm ですが、その事情は込み入っています。

巨大数論と数論を分かつのは、整数の全体を俯瞰する数論に対して個別の整数に着目する巨大数論という視点の違いであり、巨大数論を数論の下位分野として説明することはできません。そのことは、個別の数に命名するか命名しないかという文化の違いにも表れていますし、数論では問題にならない性質を変えない程度の細かな定義の違いが巨大数論においては問題にされるということにも表れています。

巨大数論は、現実から離れた巨大な数を扱うことによる応用数学としての意味の薄さと、それぞれの巨大な数に注目することによる純粋数学としての意味の薄さという二律背反に苛まれており、可能な限り大きな数を単純に定義するという工学にも近い性質を帯びていながらも工学が持つべき実用性を持たず、時には大きい数を定義することを目指さない美しい定義を独自性でもって評価するという芸術にも近い性質を帯びていながらも芸術が持つべき自由奔放性を持たない、既存の学問から見捨てられた世界であり、それが故の独自の魅力を有しています。

巨大数の大きさを比較により決定するのは巨大数論の重要なトピックであり、大きさを比較しやすい巨大数が研究対象として好まれ、大きさを比較しにくい巨大数は研究対象として敬遠されます。大きさを比較しやすい巨大数としては、冪乗や矢印表記などのような明快な関数により定義された巨大数があります。大きさを比較しにくい巨大数としては、 Harvey Friedman が考案した関数により定義された巨大数や、ビジービーバーのような計算不能関数により定義された巨大数などがあります。

日本における巨大数論は 2002 年頃に生まれた未成熟な学問であり、その黎明期において大量の間違いが入り込んでおり、その影響は今なお残存していますが、議論が積み重ねられ、 P.N. フィッシュさんが 2013 年に『巨大数論』を上梓し、 2018 年頃より P.N. p 進大好き bot さんが厳密な議論により大多数の間違いを排除し、その後には厳密な議論が重視されるようになっており、その水準は次第に向上しています。

巨大数論に関する本は、『巨大数論』（フィッシュ）や、『巨大数』（鈴木真治）や、『数学セミナー 2019 年 7 月号　おおきな数』（日本評論社）や、『現代思想 2019 年 12 月号　巨大数の世界』（青土社）などがあります。

## Dekatlas の開発環境

Dekatlas をビルドするためには、以下の手順に従ってください。

1. Rocq をインストールする。
2. Make をインストールする。
3. `build_makefile.sh` を実行する。
4. `build.sh` を実行する。
